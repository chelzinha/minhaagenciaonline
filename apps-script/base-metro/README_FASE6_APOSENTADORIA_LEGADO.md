# Fase 6 — Aposentadoria controlada do CRM legado

Este pacote mantém o backend oficial e acrescenta a desativação reversível do menu/sidebar CRM Lateral antigo da planilha principal.

## Ordem
1. Execute `previewAposentadoriaLegadoCrmFase6()`.
2. Confirme que `ok: true`.
3. Execute `aposentarCrmLateralFase6()`.
4. Execute `auditAposentadoriaLegadoCrmFase6()`.

## Reversão
Execute `restaurarCrmLateralFase6()`. Os triggers antigos não são reinstalados automaticamente.
