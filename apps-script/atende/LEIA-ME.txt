ATENCAO SENSIVEL
Este modulo processa dados pessoais e operacionais de postagens.
Nao salvar tokens, senhas, cookies, headers sensiveis ou payload completo em logs.

FUNCOES PRINCIPAIS
- setupInicial(): cria/valida planilha, abas e cabecalhos.
- buscarDados(): mantem compatibilidade com o front atual.
- processarAtendimentos(jsonString): importa JSON de atendimentos/resumo.
- processarEBuscar(jsonString): importa JSON de objetos captados.
- doGet(e): entrega o painel ou retorna JSON com action=dados.
- doPost(e): endpoint protegido para importacoes futuras.

PROPRIEDADES DO APPS SCRIPT
- SPREADSHEET_ID: ID da planilha operacional, quando ja existir.
- INGEST_TOKEN: token obrigatorio para importacao via doPost.

TESTE MINIMO
1. Execute setupInicial().
2. Abra o Web App do /atende e confirme que a tabela carrega.
3. Importe um JSON de atendimentos.
4. Importe um JSON de objetos captados.
5. Reimporte os mesmos JSONs e confirme que nao duplica objetos.
6. Confira README.md e TESTES.md para a validacao completa.
