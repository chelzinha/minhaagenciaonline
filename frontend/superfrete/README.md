# Minhas Postagens — Portal do Cliente v1

Rota: `/superfrete`

Inclui:
- Login de cliente via `sfClientLogin`.
- Dashboard com logo/nome do cliente, saldo, limite e disponível.
- Aba Cotação com CEP, peso em gramas, medidas, AR/MP e valor declarado.
- Aba Etiqueta com destinatário, busca CEP, declaração de conteúdo e emissão oficial.
- Histórico com SRO clicável, PDF oficial e Etiqueta AGF por overlay.
- Financeiro com conta corrente, lançamentos e cobranças/pagamentos Pix já registrados.

Observação: a aba Financeiro exibe cobranças Pix existentes. A integração de geração de QR Code Pix por provedor bancário/InfinitePay continua como módulo financeiro posterior.
