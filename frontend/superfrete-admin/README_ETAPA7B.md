# Etapa 7B — Etiqueta AGF por overlay do PDF oficial

Nesta versão, a etiqueta AGF não é mais redesenhada em HTML.
O fluxo usa o PDF oficial da SuperFrete como matriz visual e aplica uma camada branca sobre a marca SuperFrete, inserindo a logo/nome do cliente no mesmo espaço.

Arquivos principais:
- index.html
- etiqueta-overlay.html
- js/app.js
- js/config.js
- js/api.js

O PDF oficial original continua disponível como fallback obrigatório.
