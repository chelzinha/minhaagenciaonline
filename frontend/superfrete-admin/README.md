# AGF SuperFrete Admin — Etapa 3

Painel administrativo com cadastro, conta corrente e emissão administrativa simulada.

## Rotas/telas

- Login admin
- Dashboard
- Clientes
- Emitir etiqueta
- Cadastro
- Conta corrente

## Como usar a emissão simulada

1. Configure a URL do Web App.
2. Entre com as credenciais administrativas configuradas no backend. A senha não deve ficar salva no HTML.
3. Abra **Emitir etiqueta**.
4. Registre uma recarga SuperFrete simulada para alimentar a carteira espelho.
5. Escolha cliente/remetente.
6. Preencha destinatário, pacote, valores e itens da DC-e.
7. Clique em **Gerar etiqueta simulada**.

Nenhum pedido real é criado na SuperFrete nesta etapa.
