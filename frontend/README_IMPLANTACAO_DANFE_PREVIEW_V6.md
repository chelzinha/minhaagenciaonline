# Minhas Postagens — Prévia DANFE Simplificado Etiqueta 10x15

## Modo atual

Este pacote habilita apenas o modo de amostragem interna. Toda impressão contém a marca d'água:

`PRÉVIA DE TESTE — NÃO UTILIZAR PARA TRANSPORTE`

O PDF importado continua preenchendo os dados operacionais da postagem. A prévia fiscal é um fluxo separado de conferência.

## Fluxo do usuário

1. Entrar no `/app` ou `/superfrete`.
2. Chegar à Etapa 2 — Dados da etiqueta.
3. Importar o PDF da NF-e.
4. Revisar os dados preenchidos da postagem.
5. Clicar em **Revisar prévia DANFE 10x15**.
6. Corrigir campos fiscais na tela de conferência, quando necessário.
7. Registrar como **conferida** ou **com divergência**.
8. Imprimir somente para amostragem interna.

## Auditoria administrativa

Acesse `/superfrete-admin/danfe-auditoria.html` ou use o item **Amostragem DANFE** no menu do admin.
